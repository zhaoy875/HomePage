## backbone中model和collection合并的underscore函数
------------------------------------------------
### Model
* **keys**
```javascript
  _.keys({one: 1, two: 2, three: 3});
    => ["one", "two", "three"]
```
* **values**
```javascript
  _.values({one: 1, two: 2, three: 3});
=> [1, 2, 3]
```
* **pairs**
* **invert**
* **pick**
* **omit**
